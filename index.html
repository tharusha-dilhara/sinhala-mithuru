Sinhala Mithuru

A research-driven adaptive learning ecosystem for primary Sinhala education, combining handwriting analysis, pronunciation feedback, reading comprehension, and grammar correction in one learning pathway.

Built for Grades 1-5 with a focus on low-resource Sinhala language constraints, classroom data collection, and child-centered feedback loops.

Research domain

A focused response to fragmented Sinhala learning tools.

Sinhala Mithuru addresses a gap in early Sinhala learning support: most tools solve one isolated task, while primary learners need connected feedback across writing, listening, speaking, reading, and grammar.

01

Fragmented learning support

Existing tools often focus on a single skill, leaving teachers and children without a unified progress model.

02

Missing process feedback

Handwriting tools usually evaluate the final image, not stroke order, pen movement, and formation behavior.

03

Low-resource language constraints

Sinhala-specific child language datasets are limited, especially for pronunciation and grammar correction.

Research objectives
  1. Develop a Hybrid CNN and DTW model for real-time, process-oriented Sinhala handwriting analysis.
  2. Build an auditory-vocal training module using Wav2Vec2-XLS-R for pronunciation assessment.
  3. Implement SBERT-based semantic analysis for reading comprehension and interactive storytelling.
  4. Engineer a transformer-based grammar correction model with programmatic data augmentation.
Methodology

Four AI pipelines, one learning experience.

The system is structured around specialized models that return lightweight feedback to the learner interface. Each component is designed to be measurable, teachable, and suitable for young Sinhala learners.

edit

Handwriting Analysis

Hybrid CNN and DTW

Captures stroke trajectories and compares formation patterns to support early letter writing practice.

mic

Audio-Vocal Training

Wav2Vec2 ASR and phoneme scoring

Connects listening and speaking tasks so children receive pronunciation feedback with clear learning cues.

menu_book

Interactive Narratives

LLM and SBERT semantic analysis

Supports reading comprehension and storytelling through grade-aware prompts and answer evaluation.

spellcheck

Contextual Grammar

Transformers and data augmentation

Targets common child-specific Sinhala writing errors with context-aware correction suggestions.

Architecture

A decoupled research prototype built for iteration.

The learner client captures handwriting, audio, and reading interactions. FastAPI routes inputs to specialized Python inference services, then returns structured feedback for the learning interface.

Field research

Classroom data collection across partner schools.

Field visits grounded the system in real learner behavior. The project collected handwriting, speech, and reading interaction data from primary school settings.
